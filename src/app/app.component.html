Nested menu

Pulsa en el botón para abrir el menú.

Notas

El menú está adaptado para dispositivos móviles a partir de 768px de ancho de pantalla.

El componente NestedMenu tiene dos inputs configurables: menuItems, en el cual se definen las opciones (datos) del menú; y closingTimeout, en el cual se define el tiempo (en milisegundos) que pasa desde que el ratón deja de estar encima de una opción del menú hasta que este se cierra (para dar tiempo de margen al usuario a navegar por el menú sin cierres inesperados. Por defecto, 500 ms).

El componente NestedMenu también tiene un output, itemClicked, el cual define el comportamiento de cada opción del menú al ser pulsada. Es una función que se llama cada vez que se pulsa una opción del menú, pasando por parámetro el id del elemento pulsado. Actualmente, se usa una función que muestra en la consola el elemento pulsado.
